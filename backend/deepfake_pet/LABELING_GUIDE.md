## 一、環境安裝（只需要做一次）

```bash
pip install matplotlib Pillow numpy
```

> Windows / Mac / Linux 都適用，**不需要** dlib 或 OpenCV。

---

## 二、準備動物圖片

挑一張動物的「**正臉照**」，要求：
- 臉朝正面（不要側臉、斜臉）
- 臉夠大、清楚
- 建議解析度 600×600 以上
- 格式：jpg 或 png

把圖片命名為 `{動物名稱}_template.jpg`，例如：
- `cat_template.jpg`
- `rabbit_template.jpg`
- `hamster_template.jpg`

---

## 三、下載標點工具

從 Brian 取得 `label_tool.py`，放到和圖片**同一個資料夾**。

---

## 四、執行工具

```bash
python label_tool.py cat_template.jpg
```

視窗打開後就可以開始標點。

---

## 五、怎麼標點

### 標記模式（0～67 點）

**左鍵點擊** = 新增下一個點（照順序，不能跳）
**右鍵點擊** = 撤回最後一個點（點錯了用這個）

畫面上方黃色文字會告訴你現在在標哪個部位，照著提示標即可。

## 七、調整點位（標完 68 點後）

68 點標完後進入**調整模式**：

- **左鍵點綠色點** → 選中（變紅）
- **左鍵點新位置** → 把選中的點移過去
- **右鍵點任意處** → 取消選取
- **S 鍵** → 存檔

反覆調整到位置合理為止。

---

## 八、存檔

按 **S 鍵** 存檔，會在圖片旁邊產生同名 JSON：

```
cat_template.jpg          ← 你的動物圖片
cat_template_landmarks.json  ← 自動產生的標點資料
```

## 常見問題

**Q：視窗打不開？**
Windows 先裝 Tk：`pip install tk`
Linux：`sudo apt install python3-tk`

**Q：點標歪了怎麼辦？**
- 標記中：右鍵撤回，重新點
- 標完後：進調整模式選點再移位置

**Q：JSON 的格式長什麼樣？**
```json
[[123.4, 456.7], [130.1, 460.2], ...]
```
共 68 個 `[x, y]`，對應到第 0～67 號特徵點。
